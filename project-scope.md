# AI-Powered Ticket Management System

## Problems

We received hundreds of support emails daily. Our agents manually read, classify, and responde to each ticket - which is slow and leads impersonal, canned responses.

## Solutions

Build a ticket management system thet uses AI to automatically classify, respond to, and route support tickets - delivering faster, more personalized responses to students while freeing up agents for complex issues.

## Ticket Statuses

- **Open** – newly created, awaiting agent action
- **Resolved** – agent has responded and considers the issue handled
- **Closed** – ticket is fully complete and no further action is needed

## Ticket Categories

- General Question
- Technical Issue
- Refund Request

## User Roles

- **Admin** – seeded at deployment; can create and manage agent accounts
- **Agent** – created by admin; handles day-to-day ticket work

## Features

- Received support emails and create tickets
- Auto-generate human-friendly responses using a knowledge base
- Ticket list with filtering and sorting
- Ticket detail view
- AI-powered ticket classification
- AI summaries
- AI-suggested replies
- User management (admin only)
- Dashboard to view and manage all tickets
